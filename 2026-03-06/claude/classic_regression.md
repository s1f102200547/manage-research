統計回帰
├─ A. 目的変数の型による分類
│  ├─ 連続値 → 線形回帰、多項式回帰、リッジ、ラッソ...
│  ├─ 二値（0/1） → ロジスティック回帰
│  ├─ カウント（0,1,2...） → ポアソン回帰、負の二項回帰
│  ├─ 順序（低<中<高） → 順序ロジスティック回帰
│  ├─ カテゴリ（A/B/C） → 多項ロジスティック回帰
│  └─ 生存時間（イベントまでの時間） → Cox比例ハザード回帰
│
├─ B. 説明変数と目的変数の関係による分類
│  ├─ 線形関係を仮定
│  │  ├─ 単回帰（説明変数1つ）
│  │  ├─ 重回帰（説明変数複数）
│  │  └─ 一般化線形モデル（GLM）：リンク関数で非正規分布に拡張
│  ├─ 非線形関係を仮定
│  │  ├─ 多項式回帰
│  │  ├─ スプライン回帰
│  │  └─ GAM（一般化加法モデル）
│  └─ 関係の形すら仮定しない
│     └─ ノンパラメトリック回帰（カーネル回帰、LOESS）
│
├─ C. 推定上の問題への対処による分類
│  ├─ 多重共線性 → リッジ回帰（L2正則化）
│  ├─ 変数選択 → ラッソ回帰（L1正則化）
│  ├─ 両方 → Elastic Net
│  ├─ 外れ値 → ロバスト回帰（Huber、M推定）
│  ├─ 内生性（交絡） → 操作変数法（2SLS）
│  └─ 欠損・打ち切り → トービットモデル、Heckmanモデル
│
├─ D. データ構造による分類
│  ├─ 独立な個体 → 通常の回帰
│  ├─ パネル（個体×時間） → 固定効果・ランダム効果モデル
│  ├─ 階層（生徒∈学校∈地域） → 混合効果モデル（HLM/マルチレベル）
│  ├─ 時系列 → AR、ARIMA、VAR
│  └─ 空間 → 空間回帰（SAR、SEM）
│
└─ E. 推定のパラダイムによる分類
   ├─ 頻度主義（最小二乗法、最尤推定法）
   └─ ベイズ推定（事前分布 + 尤度 → 事後分布）

推奨配分（エンジニア志向の場合）
├─ 統計回帰の概念的基盤: 50%  ← ここが最重要
├─ ML回帰の手法と使い分け: 25%
├─ 深層学習回帰: 15%
└─ AI回帰（LLM統合等）: 10%  ← 今まさに動いている領域
