# 学習順（数学的原理が簡単 → 難しい）

## ① 回帰分析系

**偏相関・回帰分析・重回帰・ロジスティック回帰**

理由

* 線形代数 + 微分だけ
* ほぼすべての統計・MLの基礎

理解できる概念

* 相関
* 最小二乗
* 確率
* 勾配

CS例

```
クリック率 = a × 表示回数 + b
```

最も基本。

---

## ② ベイズ

**ベイズ推定**

理由

* 確率の再計算だけ
* 数学は高校レベル

理解できる概念

* 事前確率
* 更新
* 不確実性

CS例

スパム判定

```
P(Spam | word)
```

---

## ③ クラスタリング

（K-means）

理由

* アルゴリズムが非常に直感的
* 距離だけ

CS例

```
ユーザーを3グループに分ける
```

## ④ 次元削減

（PCA）

理由

* 線形代数
* 可視化で重要

* 分散

CS例

```
ユーザーデータ100次元 → 2次元
```
